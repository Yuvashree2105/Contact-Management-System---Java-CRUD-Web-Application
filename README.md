Contact Management System

This project is a Java-based Contact Management System developed during my internship learning phase, mainly focused on backend development and database operations. The application performs CRUD (Create, Read, Update, Delete) functionalities for managing contact records efficiently.
The frontend/UI was used for interaction, while my primary learning and implementation focus was on backend logic, database handling, and server-side development.

Features

Add new contacts
View stored contacts
Update contact information
Delete contacts
Database connectivity with PostgreSQL
Backend request handling
CRUD operations using JDBC

Tech Stack

Java
JDBC
PostgreSQL
Apache Tomcat
Servlets
SQL

My Focus / Contribution

Backend logic development
Database connectivity using JDBC
Writing SQL queries
CRUD implementation
Server-side handling with Java Servlets
Understanding deployment in Tomcat

Learning Outcome

This project helped me strengthen my backend development knowledge, database handling, and Java web application concepts.
